강아지상 vs 고양이상, 나는 어떤 스타일일까?

연예인들의 얼굴을 보며 흔히 '멍뭉미 넘치는 강아지상' 이라거나 '도도한 매력의 고양이상' 이라는 표현을 많이 씁니다. 그렇다면 나의 얼굴 특징은 어떤 동물상에 더 가까울까요? 이제 AI 기반 얼굴 분석 테스트를 통해 쉽고 빠르게 확인해보세요!

테스트 방법 안내

  1. 상단의 '사진 업로드하기' 버튼을 클릭하여 본인의 얼굴이 잘 나온 사진을 선택합니다. 정면을 응시하고 이목구비가 뚜렷한 사진일수록 분석 정확도가 높아집니다.
  2. 사진이 업로드되면, 미리 학습된 Google Teachable Machine 기반의 인공지능 모델이 즉시 이미지 분석을 시작합니다.
  3. 수 초 내에 강아지상과 고양이상의 확률이 퍼센트(%)로 화면에 표시됩니다. 더 높은 수치를 기록한 쪽이 당신의 대표적인 관상 특징입니다.
  4. 업로드된 사진은 오직 본인의 브라우저 내에서만 처리되며, 외부 서버로 전송되거나 저장되지 않으므로 안심하고 테스트를 즐기셔도 됩니다.

대표적인 동물상 특징 요약

  • 🐶 강아지상 특징: 선한 눈매, 처진 눈꼬리, 둥글고 부드러운 얼굴형을 가지며 친근하고 순둥순둥한 매력을 발산합니다. 보는 사람에게 편안함과 귀여움을 주는 인상입니다. (대표 연예인: 박보영, 수지 등)
  • 🐱 고양이상 특징: 날렵한 턱선, 살짝 올라간 눈꼬리, 크고 뚜렷한 눈망울을 가지며 도도하고 세련된 매력을 풍깁니다. 차가워 보이지만 동시에 매혹적이고 신비로운 인상입니다. (대표 연예인: 한소희, 뉴진스 해린 등)

결과는 재미와 흥미를 위한 목적으로 제작되었으니, 친구들과 결과를 공유하며 즐거운 시간을 보내보세요!
